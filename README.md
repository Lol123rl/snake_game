Cube Snake is a 3D snake battle game where you move around the sides of a cube instead of just playing on a flat board.
The goal is to eat food, grow bigger, complete the level task, and avoid crashing into snake trails.
Each cube face has a different color and theme, like snow, beach, lava, forest, space, and home.
You can move off one side of the cube and appear on another side, which makes the game feel more like a real cube.
There are enemy snakes that move around, collect food, and try to cut you off.
Some levels ask you to get points, some ask you to kill snakes, and some ask you to capture cube faces.
To capture a face, you need to eat two foods on that face, and then it turns green.
There are also power-ups like shield, ghost, boost, and bomb to make the game more interesting.
Level 6 has a boss snake with extra lives, so it is harder than the normal levels.
The controls are the arrow keys to move, Enter to start or retry, and R to go back or replay after a level.
This game was made as a simple Godot project, so the code is all in one script to make it easier to edit and understand.
